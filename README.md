Bu proje, kapsamlı bir -Eğitim Kampı- sürecinde, modern kurumsal web uygulamalarının yönetim paneli (Admin Dashboard) ihtiyaçlarını ve API entegrasyon süreçlerini öğrenmek amacıyla geliştirilmiştir. Eğitim boyunca asenkron programlama ve katmanlı mimari standartları pratik edilmiştir.


🛠️ Teknik Yetkinlikler (Tech Stack)
Backend: C#, ASP.NET Core 8.0 MVC

Veritabanı: MS SQL Server, Entity Framework Core (Code First)

Mimari: N-Tier Architecture (DataAccess, Business, Entity, UI Layers)

Frontend: HTML5, CSS3, Bootstrap 4/5, JavaScript

Görselleştirme: Chart.js (Dinamik Şehir & To-Do İstatistikleri)

Araçlar: ViewComponents, DTO (Data Transfer Objects), AutoMapper

✨ Temel Özellikler
Dinamik Dashboard: Veritabanından gelen anlık verilerle (siparişler, müşteriler, mesajlar) beslenen canlı panel.

Gelişmiş Grafikler: Şehirlere göre müşteri dağılımı ve yapılacak işlerin durum takibini sağlayan dairesel grafikler.

Dinamik Sidebar & Mesajlaşma: SQL üzerinden yönetilen dinamik menü yapısı ve son mesajlar sistemi.

ToDo List: AJAX destekli, durum güncellenebilir görev yönetim modülü.
